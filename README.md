# Test Kitchen

TestKitchen is a [MediaWiki](https://www.mediawiki.org/) extension that provides dynamic stream configs
to [EventStreamConfigs](https://wikitech.wikimedia.org/wiki/Event_Platform/Stream_Configuration).

For information on how to install and use this extension, please see: https://www.mediawiki.org/wiki/Extension:TestKitchen

	cd extensions
	git clone https://gerrit.wikimedia.org/r/mediawiki/extensions/TestKitchen.git
